Voice-note transcription uses Groq when you provide your own API key — it's noticeably faster than the Worker fallback, and each teammate has their own free-tier quota (8 hours of audio/day).
How do I get a free Groq key?
  1. Sign up at console.groq.com (free, no card needed)
  2. Open API Keys → Create API Key
  3. Copy the gsk_... string and paste it above
  4. Click Test & save
Your key is stored in this browser only — never sent to our Worker or to other teammates.
